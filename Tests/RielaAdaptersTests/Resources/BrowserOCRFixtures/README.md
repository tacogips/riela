# Browser OCR Fixtures

These PNG fixtures were rendered in Brave Browser from
`tmp/ocr-browser-fixtures/page.html` and captured via Computer Use-guided browser
interaction plus macOS screenshot cropping.

- `japanese_meeting.png`: Japanese source text for Japanese to English OCR translation.
- `english_library.png`: English source text for English to Japanese OCR translation.
- `chinese_ticket.png`: Simplified Chinese source text for Chinese to English OCR translation.
